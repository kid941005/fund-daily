## 技术实现细节

### 前端组件结构
```vue
<!-- 清仓按钮 -->
<button class="btn-danger" @click="confirmClearAll" v-if="store.holdings?.length">清仓</button>

<!-- 确认对话框 -->

      </ul>

    <div class="modal-actions">
```

### 后端API流程
```
前端点击清仓 → 显示确认对话框 → 用户确认 → 调用clearHoldings()
→ POST /api/holdings/clear → 检查用户登录 → 调用db.clear_holdings()
→ DELETE FROM holdings WHERE user_id = ? → 返回成功响应
```

### 数据库操作
```python
def clear_holdings(user_id):
    """清空用户的所有持仓"""
    with get_db() as conn:
        with get_cursor(conn) as cursor:
            cursor.execute("DELETE FROM holdings WHERE user_id = %s", (user_id,))
            conn.commit()
```

## 安全考虑

1. **用户认证**：清仓操作需要用户登录，未登录用户无法执行
