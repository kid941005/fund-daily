# 默认凭据

## 管理员账户

项目初始化后会创建一个默认管理员账户：

- **用户名**: `admin`
- **密码**: `admin123`
- **用户ID**: 自动生成（每次初始化不同）

## 安全注意事项

⚠️ **重要**: 默认密码 `admin123` 仅用于开发和测试环境。

**生产环境必须：**

1. **立即修改密码** - 首次登录后立即更改密码
2. **使用强密码** - 至少12位，包含大小写字母、数字和特殊字符
3. **考虑禁用或删除默认账户** - 创建新的管理员账户后禁用或删除此默认账户

## 如何修改密码

1. 登录系统
2. 访问 `/api/change-password` 端点
3. 提供旧密码和新密码

## 如何创建新的管理员账户

1. 注册新账户
2. 根据需要手动提升权限（如果系统支持角色管理）

## 迁移脚本

默认管理员账户由以下迁移脚本创建：
- `scripts/migrations/0025_create_admin_user.py`
- `scripts/create_admin_user.py`（手动执行版本）
